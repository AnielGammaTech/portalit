3. Helper functions `readCookie(name)`, `writeCookie(name, value)`, `deleteCookie(name)`:
   - `readCookie`: splits `document.cookie` on `/;\s*/`, checks for prefix `name + '='`, returns decoded value or null. SSR safe (`typeof document === 'undefined'`).
   - `writeCookie`: sets `document.cookie = "name=encoded; Path=/; Max-Age=<MAX_AGE_SECONDS>; SameSite=Lax"`. If not localhost, appends `Domain=.gtools.io` and `Secure`.
   - `deleteCookie`: sets Max-Age=0 with the same domain attrs.

4. Helper functions `readFromCookie(key)`, `writeToCookie(key, value)`, `clearCookieChunks(key)`:
   - `readFromCookie`: reads meta cookie, if value starts with `chunks:N` reassembles from `${key}.0` through `${key}.${N-1}`, returns assembled string or null. If any chunk is missing, return null (corrupted → treat as no session). If meta doesn't start with "chunks:" treat as legacy single-value cookie.
   - `writeToCookie`: always `clearCookieChunks(key)` first. If value length ≤ CHUNK_SIZE, write single cookie. Else compute `chunks = Math.ceil(value.length / CHUNK_SIZE)`, write meta cookie `chunks:N`, then write chunks `${key}.0`..`${key}.${chunks-1}`.
   - `clearCookieChunks`: delete meta + up to 10 chunk indices (loop i=0..9 deleteCookie).

5. Exported factory `createAuthStorage()`:
```javascript
export const createAuthStorage = () => ({
  getItem(key) {
    if (IS_LOCALHOST) {
      return window.localStorage.getItem(key);
    }
    // Cookie first (post-rollout), fallback to localStorage (backcompat D-18)
    const fromCookie = readFromCookie(key);
    if (fromCookie != null) return fromCookie;
    return window.localStorage.getItem(key);
  },

  setItem(key, value) {
    if (IS_LOCALHOST) {
      window.localStorage.setItem(key, value);
      return;
    }
    // Write both during backcompat window (D-18). Removing the localStorage
    // write is a deferred followup (~1 week post-phase-6).
    writeToCookie(key, value);
    try { window.localStorage.setItem(key, value); } catch { /* quota etc — ignore */ }
  },

  removeItem(key) {
    if (IS_LOCALHOST) {
      window.localStorage.removeItem(key);
      return;
    }
    clearCookieChunks(key);
    try { window.localStorage.removeItem(key); } catch { /* ignore */ }
  },
});
```

6. Also export a default export matching createAuthStorage()-produced object for callers that prefer `import authStorage from '@/lib/auth-storage'`:
